문제

아래 설명된 의미에 해당하는 용어를 쓰세요 
 한 클래스에 이름과 반환값의 자료형은 동일하지만 전달인자가 서로 다른 메서드  

 부모 클래스와 자식 클래스에 이름, 반환값의 자료형, 전달인자가 모두 동일한 메서드  

 서로 관련된 여러 클래스를 하나의 패키지로 묶어서 체계적으로 관리  

 다른 패키지 클래스를 가져오기  

 데이터와 이를 관리하는 메서드로 구성된 객체의 구조를 선언한 설계도  

 모든 클래스에서 접근할 수 있도록 허용  

 해당 클래스 내에만 접근할 수 있도록 허용  

 자식클래스가 부모클래스의 변수, 상수, 메서드를 모두 상속  

 객체 자신에 대한 레퍼런스  

 부모클래스 객체  

 클래스 객체를 메모리에 생성하며, 초기화를 위해 생성자 호출  

 메서드 종료시 제어흐름이 메서드를 호출했던 곳으로 되돌아감  

 클래스 객체의 데이터 값과 전달인자의 데이터 값을 비교하여 같으면 참을 반환하고 다르면 거짓을 반환  

 비교연산자로 두 변수값이 같으면 참을 반환하고 다르면 거짓을 반환  

 해당 객체를 문자열로 변환하는 메서드  

 조건식이 참이면 해당 명령문을 실행하는 선택문  

 앞에 제시된 조건식이 모두 거짓이고 현재 조건식이 참이면 해당 명령문을 실행하는 선택문  

 앞에 제시된 조건식이 모두 거짓이면 해당 명령문을 실행하는 선택문  

 한 번 이상 실행하는 반복문의 시작을 표현  

 조건식이 참일 때마다 해당 명령문을 실행하는 반복문
