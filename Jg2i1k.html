문제

아래 설명된 의미에 해당하는 용어를 쓰세요 
 서로 관련된 여러 클래스를 하나의 패키지로 묶어서 체계적으로 관리  

 다른 패키지 클래스를 가져오기  

 데이터와 이를 관리하는 메서드로 구성된 구조를 선언한 설계도  

 모든 클래스에서 접근할 수 있도록 허용  

 해당 클래스 내에만 변수나 메소드 등에 접근할 수 있도록 허용  

 자식 클래스나 같은 패키지내 다른 클래스에서 변수나 메소드 등에 접근할 수 있도록 허용  

 자식클래스가 부모클래스의 변수, 상수, 메서드를 모두 상속  

 객체 자신에 대한 레퍼런스  

 부모클래스 객체에 대한 레퍼런스  

 클래스 객체를 메모리에 생성하며, 초기화를 위해 생성자 호출  

 메서드 종료시 제어흐름이 메서드를 호출했던 곳으로 되돌아감  

 한 번 이상 실행하는 반복문의 시작을 표현  

 조건식이 참일 때마다 해당 명령문을 실행하는 반복문  

 초기식, 조건식, 증감식으로 구성되고, 조건식이 참일 때마다 해당 명령문을 실행하는 반복문  

 조건식이 참이면 해당 명령문을 실행하는 선택문  

 앞에 제시된 조건식이 모두 거짓이면 해당 명령문을 실행하는 선택문  

 삼항연산자에서 조건식이 참인지 거짓인지 확인하는 기호  

 정수형 변수와 특정 정수값을 비교할 때 사용되는 기호  

 0 ~ 1 사이의 실수 중 임의의 수 하나를 반환하는 메소드  

 복합대입연산자로 연산자의 왼쪽 변수값에 오른쪽 값만큼 더한 값을 왼쪽 변수 공간에 저장
